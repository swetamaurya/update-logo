# Royal Thread Craft – Logo Overlay Tool

Ek simple web tool jisse aap **photo ya video** par apna **Royal Thread Crafts logo (watermark)** laga sakte hain.

## Features
- 📷🎬 Photo aur Video dono support
- 👆 Logo ko **ungli se drag** karke kahin bhi rakho
- 🤏 **Pinch-to-zoom** (2 ungli) se chhota/bada; desktop par **Ctrl + scroll**
- 🎨 Circle ke andar **background fill** (Cream / White / Custom color) — logo ka color nahi badalta
- ✨ Opacity, Glow, BG Dark controls
- ⬇ Download: **Photo** original quality (JPG/PNG), **Video** MP4 (jahan browser support kare) logo overlay ke saath

## Deploy on Vercel
Ye ek **static site** hai — sirf `index.html`.

1. Is repo ko GitHub par push karein.
2. [vercel.com](https://vercel.com) par **New Project** → GitHub repo import karein.
3. Framework Preset: **Other** (kuch change nahi karna), **Deploy** dabayein.
4. Bas! Aapko ek public link mil jayega jise kisi ko bhi bhej sakti hain — wo browser me kholkar tool use kar lega.

Local me test karne ke liye `index.html` ko browser me khol lein.
